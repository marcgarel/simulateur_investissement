# Simulateur d'investissement avec Shiny

Cette application Shiny permet de simuler l'évolution d'un investissement en fonction :
- Du montant de départ,
- De l'épargne mensuelle,
- De la performance annuelle,
- De la durée de l'investissement.

## Fonctionnalités
- Visualisation de la valeur totale de l'investissement et des intérêts cumulés.
- Affichage des valeurs finales (investissement total et intérêts cumulés).
- Interface interactive pour ajuster les paramètres.

## Déploiement
L'application est conçue pour être déployée sur [shinyapps.io](https://www.shinyapps.io/).
